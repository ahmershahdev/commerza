# Admin CSS Modules Guide

This folder contains layered admin CSS module files.

## Files

- `01-base.css`: Variables, base typography, and reset-like rules.
- `02-layout.css`: Structural layout and spacing systems.
- `03-components.css`: Reusable admin UI components.
- `04-utilities.css`: Utility/helper classes.
- `05-responsive.css`: Responsive breakpoints and mobile adjustments.

## Notes

- Keep module load order aligned with numeric prefixes.
- Prefer editing existing modules over adding one-off inline styles.

# Frontend CSS Modules Guide

This folder contains modular storefront CSS files imported by `../style.css`.

## Files

- `base.css`: Core reset, variables, and global typography.
- `navigation.css`: Navbar, nav links, and responsive navigation states.
- `search.css`: Search input container and interaction states.
- `search-suggestions.css`: Search suggestion list/dropdown visuals.
- `carousel.css`: Banner/slider styles and transitions.
- `products.css`: Product listing, card, badge, and pricing styles.
- `layout-sections.css`: Shared layout helpers for page sections.
- `footer.css`: Footer column/link/icon styling.
- `newsletter.css`: Newsletter callout and form styles.
- `offcanvas.css`: Mobile offcanvas and overlay styles.
- `wishlist-tracking.css`: Wishlist and order-tracking specific styles.
- `page-hero-wishlist.css`: Wishlist page hero treatment.

## Maintenance

- Keep reusable styles in modules; avoid duplicating selectors across files.
- Verify responsive behavior after editing module-level layout rules.

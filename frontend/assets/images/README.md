# Frontend Images Folder Guide

This folder stores storefront images.

## Subfolders

- `favicon/`: Browser/site icon files.
- `logo/`: Brand logo assets.
- `products/`: Product and category image assets.
- `slider/`: Homepage/category slider images.
- `users/`: User-uploaded profile images.

## Notes

- User-generated images in `users/` should be validated and sanitized server-side before storage.
- Keep product and slider assets optimized for web delivery.

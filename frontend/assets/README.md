# Frontend Assets Folder Guide

This folder holds all shared storefront static resources.

## Subfolders

- `css/`: Global and modular storefront styling.
- `js/`: Shared storefront behavior, protection logic, and module orchestration.
- `images/`: Brand/product/user image assets.
- `videos/`: Hero and product video assets.

## Notes

- Keep CSS/JS modular and ordered through entry files.
- Prefer compressed media to reduce page load costs.
- Do not store secrets in frontend assets.
